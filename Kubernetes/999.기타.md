# timezone 설정

```yml
containers:
  - name: 컨테이너 이름
    image: 이미지
    imagePullPolicy: Always  # 항상 이미지 가져오기
    env:
    - name: TZ
    value: Asia/Seoul  # 타임존 설정
imagePullSecrets:
  - name: regcred
```

# 사내 리파지토리 설정

```
kubectl create secret --namespace {네임스페이스 이름} docker-registry regcred --docker-server={주소} --docker-username={유저} --docker-password={비번} --docker-email={이메일}
```
